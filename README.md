Rain Near Me
========

Find the closest raining city near you using 3 APIs- Algolia Places, GEODB Cities Near Location, and OpenWeatherMap. 


Features
--------

- AutoLocate and Autocomplete Searchbar
- 3 different API requests using Promises
- Fun GSAP Animations
